This is a web app to help guitarist better understand their instrument in different tunings.

Currently, it's very functional.

    You can
  • Tune each individual string
  • Tune all strings at once
  • Visualize different musical scales within different tunings
  • See different chords within different tunings

Right now

the UI is clunky. I plan to update this site quite often with different scales and general touch ups. So bear with me.

Make sure you clear out the scales with the reset scales button if you want to use the chords, vice versa for the chords.
